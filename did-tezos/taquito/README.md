# Blockchain interactions

## Installation

Requires Node.js and NPM.

```bash
$ npm install
```

### Tezos set up
Get a test "account" here: https://faucet.tzalpha.net/

Then activate it with
```bash
$ ./tezos-client activate account <account_name> with <faucet_result.json>
```


## Origination

```bash
$ node originate.js
```
